# Project Name: Course Project SDC320 - Playlist Application

## Project Description:
This project is a way to display my knowledge of how I can apply an interface,
an abstract class, composition, polymorphism, constructors and perform CRUD
operations using an SQLite database. I created a music playlist that allows
users to add, remove, play, and pause their music in addition to creating the
playlist and personalization.

## Project Tasks
- Taskl Week1: Create the proposal to show the plan and overview of what I
was creating. This included listing out all aspects I planned to use in the application,
as well as the purpose, output, and data storage aspects.
- Task2 Week2: Creating an outline of all aspects. This included documentation that showed
the UML Diagram, and breaking down the method, constructors, and what each part of the diagram
showed (such as composition, polymorphism, etc.) Along with that task, I had to create the plan
for the database storage and define all in the UML Diagram.
- Task3 Week3: First step of development - Class implementation is the primary objective
to begin the application and the framework. I included constructors and access specifier.
- Task4 Week4: Data implementation. Added the SQLite database support to the application
- Task5 Week5: Create the summary of the project and show a map that discusses how the project
went.
